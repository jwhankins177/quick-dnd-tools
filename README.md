# Quick D&D Tools

A lightweight web application for D&D parties to track characters and use quick tools during gameplay. Perfect for when your party has some time but doesn't have access to their usual materials - allows you to make progress and save things for later.

## Usage

Simply open `index.html` in any modern web browser. No installation or build process required!

The application uses your browser's local storage to save all data, so your characters and settings persist between sessions.

## Technologies

- Pure HTML5, CSS3, and JavaScript
- No dependencies or frameworks
- Responsive design for mobile, tablet, and desktop
- Works offline after first load

## Browser Compatibility

Works in all modern browsers:
- Chrome/Edge (recommended)
- Firefox
- Safari
- Opera

## Tips

- All data is stored locally in your browser
- To share with party members, each person can maintain their own characters
- Clear your browser data will reset the application
- Best used on the same device for consistency
